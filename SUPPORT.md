# Support

## Where to ask for help

- Questions and general usage discussion:
  `https://github.com/15680676726/superSpider/discussions`
- Bug reports and concrete feature requests:
  `https://github.com/15680676726/superSpider/issues`
- Private security reports:
  `https://github.com/15680676726/superSpider/security`

## What to include

When asking for help, include:

- the `superSpider` commit, tag, or package version
- your OS and Python version
- install method: `pip`, script install, Docker, from source, or desktop app
- the command you ran
- the error message or the unexpected behavior
- relevant logs or screenshots when available

## Scope

This repository is maintained on a best-effort basis. Public support does not
include guaranteed response times, managed hosting, or private consulting.
