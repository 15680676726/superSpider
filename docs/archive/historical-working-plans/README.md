# Historical Working Plans

This directory stores internal working plans that are no longer part of the active planning reference chain.

They are kept for engineering history, auditability, and design archaeology, but they should not stay mixed into `docs/architecture/working-plans/`.

Move a working plan here when:

- it is no longer referenced by current root planning/status docs
- it is no longer treated as an active implementation baseline
- it still has historical value and should not be deleted outright

Do not move documents here if they are still explicitly referenced by:

- `AGENTS.md`
- `TASK_STATUS.md`
- `implementation_plan.md`
- other current active specs/plans that still drive work
