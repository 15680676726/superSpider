# Public Legacy Release Notes

This directory stores pre-`superSpider` release notes that are kept only for historical reference.

Why these files were moved out of `website/public/`:

- they describe older public releases from the previous product identity
- keeping them under the website publish directory kept outdated naming and legacy capability claims publicly reachable
- current public product docs should describe the live `superSpider` system, not mixed historical branding

Rules:

- do not put these files back under `website/public/`
- when the public site needs a changelog, write it against the current `superSpider` product line
- use this archive only when historical release context is needed
